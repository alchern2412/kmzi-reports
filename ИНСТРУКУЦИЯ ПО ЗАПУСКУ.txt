-Часть лабораторных написаны на языке Java
-Часть на JS
-Часть на C#

Для запуска Java Нужно иметь на машине установленную JDK или JRE (!!!), а также любую IDE (Напр. Intellij Idea, Eclipse, NetBeans). Запускать через них.

Для запуска C# приложений - достаточно перейти в папку /Debug/bin - и запустить .exe файл. Для полноты выполнения либо различий в .NET Framework на другой машине принято решение оставить полностью Solution проекта.

Для запуска JS Приложений должна быть установлена "свежая" версия Node.js (!!!)
Нужно зайти с cmd.exe в папку проекта. 
Выполнить команду "npm install" - для того, чтобы установить используемые в лабораторной работа пакеты.
Затем командой "node index.js" запустить главный файл программы
